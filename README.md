# 📚 Menu de Favoritos Profissional

Um menu web interativo e organizado para acessar rapidamente links profissionais relacionados ao SENAC e UniCesumar.

## 🎯 Propósito

Centralizar e organizar links úteis de plataformas, documentos, cronogramas e sistemas em uma única página navegável, eliminando a necessidade de procurar por bookmarks ou histórico do navegador.

## 📋 Conteúdo

O menu está organizado em duas estruturas principais:

### SENAC
- **Administrativo**: Links para RH, cronogramas e plataformas de gestão
  - PTD: Turmas 2026
  - Acompanhamento UCs
  - TOTVS MEU RH
  - Cronograma
  - OrionWeb
  - Biblioteca
  - Plataforma Senior
- **Avaliação**: Formulários e documentos de avaliação
- **Documentação**: Legislação, regimentos e diretrizes
- **Calendários**: Calendários acadêmicos
- **TEC-ADM**: Horários e cronogramas do Técnico em Administração
- **TEC-IA**: Cronogramas do Técnico em Inteligência Artificial

### Unicesumar
Repositórios, intranet, biblioteca, sistemas de RH, plataformas de ensino e documentação.

## 🚀 Como usar

1. Abra o arquivo `acessos.html` em seu navegador
2. Clique nos itens com pasta para expandir/recolher seções
3. Clique nos links para acessar os recursos

## 💾 Estrutura

```
favorite-web/
├── acessos.html    # Arquivo principal com o menu
└── README.md       # Este arquivo
```

## 🎨 Características

- Interface limpa e moderna
- Menu com expansão/recolhimento de seções
- Ícones do Font Awesome para melhor visualização
- Design responsivo para diferentes tamanhos de tela
- Links organizados em categorias e subcategorias
